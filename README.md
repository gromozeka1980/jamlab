# 🎶 jamlab (рабочее название)

Экспериментальная ветка реляционного инструмента для импровизации. Развивается дальше; стабильная демо-версия живёт в [koto](https://gromozeka1980.github.io/koto/).

**▶ https://gromozeka1980.github.io/jamlab/**

## Идея

Клавиши задают **шаг по гамме**, а не абсолютную ноту — промахнуться мимо лада нельзя. Несколько инструментов/ладов (блюз, джаз с авто-ладом, кото, гамелан, грёза, восток, светлая), аккордо-зависимые бенды, фоновые грувы, визуализатор, выразительность через гироскоп.

Ванильный JS без зависимостей и сборки: `index.html` (разметка) + `style.css` + `app.js` (движок) + `i18n.js` (переводы). Цель — обкатать перед упаковкой в мобильное приложение (Capacitor).
